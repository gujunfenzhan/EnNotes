状语从句

状语从句

时间状语

1、当...时候:when/while/as/ (by) the time/the+(时间 n)/

例句:(by)The moment i saw him,i fell in love with him.

2、在...之前:before(翻译方式:before翻译成然后，顺序翻译)

3、在...之后:after(翻译方式:after也翻译成然后，但要从后往前翻)

(先)I had my breakfast (before(然后) i go to school.)

我先吃早饭，然后去学校

before在句中的时候翻译成然后顺序翻译

I go to school after(然后) (i had my breakfast)(我吃了我的早饭).

我先吃早饭，然后去学校

4、until(到...为止)/till(到...之前)

he was a teacher (until now).

到此之前，他是个老师。(在此之后他不是了)

5、在这段时间之前:until recently

6、在现在之前:until now

7、not...until:在...之前,...不.../ 直到...,....才....

until now he is not a teacher.

在此之前，他不是老师=直到现在，他才是个老师

地点状语从句

1、在...的地方:where

注意:如果 n(地点) where +句子 ，那么他就是定语从句，如果跟n无关就是状语

I fell off where there are many snails.

我在有很多蜗牛的地方摔倒了.

I fell off form the stairs where there are many snails.(where...snails做定语)

我摔倒在了有很多蜗牛的楼梯上.

原因状语从句

1、因为:because/as/since/in that/given (that)/seeing (that)/considering (that) +句子

目的状语从句

1、为了: so that/in order that/in the hope that = (because we want)

2、以防: in case that/for fear that/lest = (because we don't want)

we shuould work hard (为了: so that) = (because we want)we can pass the test.

we should work hard(以防:lest)=(because we don't want) we cannot pass the test.

we should stop smoking (以防:lest) = (because we don't want) we will die early.

故以防后面不一定全是否定句

结果状语从句

1、所以: so that

主干,(so that)we can pass the test

2、如此...以至于...:so...that...

so.....(前面表示原因)//(后面表示结果)that....

Yj is so smart (结果状语从句)that we all like her.

Yj如此聪明以至于我们都喜欢她

让步状语从句

1、尽管A但B:although...A....,..B...

尽管(although)....A...(这里不能加but),....B.......

不能加but的原因:although是从句连词，but也是并列连词

A和B相反

遇到although找到逗号，逗号后面的是主句

2、等同于although的还有while/though/even if/even though

条件状语从句

位置:可以在主干前也可以在主干后

如果/只要/只有:If/as long as/so long as /only if

if/as long as/so long as A,B

A是B的条件之一

祝愿/推测/假设:probably/perhaps/possibly/assuming that

(if/probably/perhaps/possibly/assuming that)+虚拟语气(与事实相反)

虚拟语气的识别方式:当以上几组单词后面的动作表示过去(ved)，那么就需要反着看

only if A , B

A是B的唯一条件，除A条件之外都否定B

例:Only if 下雨,不上课

只有下雨才不上课，地震上课，世界末日也上课

方式状语从句

正如/就像:as/as if

as和as if的区别:as是真实的,as if 是虚拟的

you should do the homework as i told you.(方式/真实)

He smiles as if he were a dog.(比喻/虚拟)

I felt depressed as if it (should) rain in my heart.

比较状语从句

...非常...就像... : ...as...as...

翻译:as(非常)+as(就像...一样)

例句:She is as beautiful as I.

她非常美丽，就像我一样

...比...更... : ...more.... than...

翻译:than后面到句末放到more前面翻译

例句:She is more beautiful than I.

她比我更漂亮
