特殊句型

特殊句型

强调句

本质:加词

It is/was (被强调成分) that/which/who/whom (剩余成分位置不用换)

注:不能强调谓语动词

I love you.

强调I : It is I that love you.

强调you : It is you that i love.

对love强调 : I (do/indeed) love you.

识别:

第一种方式:去掉It is ... that句子完整就是强调句，不完整就不是

第二种方式:如果It is 后面跟的名词,that后面跟的动词，那就是强调句

It is (介词+n 状语) that (主+谓)

处理方式：删掉It is 和 that

倒装句

完全倒装句

(主 系 表)变成(表 系 主)

(主 谓 状)变成(状 谓 主)

例:

A boy is there.

There is a boy.

A boy is in the room.

In the room is a boy.

A book is on the desk/there.

On the desk/there is a book.

A boy (who wears a hat) is there. (头重脚轻)

There is a boy (who wears a hat).

只要是主系表结构就可以变成系主表

完全倒装的原因:防止头太重

部分倒装句

结构:被强调的修饰+情态动词/助动词+主谓宾

第一组 否定副词有:never/barely/rarely/hardly/little/few...

He will(情态动词/助动词) never(否定副词) like you.

Never will he like you.

处理方式:否定副词和(情态动词/助动词)都放到谓语前面

第二组 so+adj/adv

He (does) run so fast that we cannot see him.

So fast does he run that we cannot see him.

处理方式: so+adv/adj 放到谓语动词后，(情态动词/助动词)依然放在谓语动词前面

第三组 Only+状语

I ( did) realized that i like him only then.

Only then did i realize that i like him.

处理:不换顺序
